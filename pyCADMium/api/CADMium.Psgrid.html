Psgrid¶

class CADMium.Psgrid(NP, NM, a, L, loc)[source]¶

Bases: object

Generates spheroidal grid

Parameters

  • NP (int) – Number of points per integration block

  • NM (list) – Number of angluar/radial blocks

  • a (float) – half bond lenght

  • L (float) – spheroidal box size

  • loc (np.ndarray) – stencil required for derivatives
